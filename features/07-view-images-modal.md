## Funcionalidade.
**Visualizar imagens em modal de tela cheia.**

## User Story.
Como cliente,<br>
Quero visualizar as imagens de um produto em tela cheia,<br>
Para observar os detalhes com mais clareza antes de tomar uma decisão.<br>
